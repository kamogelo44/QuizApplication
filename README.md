# Quiz Application

A Java console application that tests your knowledge with multiple-choice questions.

## What This Program Does

- Presents 12 multiple-choice questions across 4 technical categories
- Tracks your score as you answer each question
- Validates your input to ensure you enter only A, B, C, or D
- Shows your final score at the end of the quiz
- Uses Object-Oriented Programming with a custom Question class

## How to Run

1. Make sure you have Java installed
2. Open the project in your preferred code editor
3. Locate and run the QuizApplication.java file
4. Follow the instructions on screen
